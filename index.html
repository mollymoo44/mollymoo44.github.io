About My Website

This is my website that I made using 'sublime text 3' on my computer. It uses a html document for the overall coding and a css stylesheet for styles. I uploaded the website using a github repository and forwarding from a free domain bought on freenom.
I will use this website to show my interests such as my favourite films, games and music as well as photos taken by me and
random things I like. I have made many websites in the past but this one will be more modern and sleek looking and uses
hover filters that will work on a computer or laptop only as you can not hover on a touchscreen device.
